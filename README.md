# DeskFit — Fitness Coaching for Office Workers & Remote Professionals

A modern, responsive single-page website for **DeskFit**, an online fitness coaching service designed specifically for office workers and remote professionals suffering from back pain, poor posture, and low energy.

## Overview

DeskFit delivers personalized fitness coaching built for busy professionals who don't have time for the gym. Our science-backed, 20-minute daily routines help you beat back pain, reclaim your energy, and get fit — all from your desk or home.

## Features

### 10 Complete Page Sections

1. **Hero** — Compelling headline, supporting sub-headline, key visual, and primary CTA
2. **Pain Points** — 3 problem cards addressing back pain, low energy, and lack of time
3. **Solution** — 3-step visual process (Assess → Plan → Move) with feature highlights
4. **Coach Profile** — Fictional example bio with credentials and coaching philosophy
5. **Program Overview** — 3 pricing tiers (Starter, Transform, Elite) with feature comparison
6. **Client Transformations** — 2 fictional before/after stories with results and testimonials
7. **Testimonials** — Sample media mentions and credible entity types
8. **FAQ** — 4 expandable accordion questions
9. **Final CTA** — Prominent call-to-action section reinforcing urgency and value
10. **Footer** — Social links, privacy policy, contact email, copyright

### Technical Specifications

- **Single HTML File** — Inline CSS and JavaScript, no external build tools required
- **Fully Responsive** — Mobile-first design works seamlessly on phones, tablets, and desktops
- **Sticky Navigation** — Smooth scroll navigation with sticky nav bar linking to key sections
- **Accessibility** — Alt text on images, semantic HTML tags, sufficient color contrast
- **SEO Optimized** — Title tag, meta description, structured content
- **Smooth Animations** — Scroll-reveal effects and interactive elements

### Design System

**Color Scheme:**
- Forest Green: `#2D5A3D` (primary)
- Off-white: `#F5F3EE` (background)
- Charcoal: `#2B2B2B` (text)

**Typography:**
- Headlines: Poppins (bold, rounded sans-serif)
- Body Text: Inter (clean sans-serif)

**Brand Tone:**
Motivating, empathetic, science-backed — like a knowledgeable friend, not a drill sergeant.

## Important Notes

All coach bios, client stories, and testimonial mentions are **FICTIONAL PLACEHOLDER CONTENT** for design purposes only. They are clearly labeled with warning badges to prevent confusion with real endorsements.

Before publishing to production, please replace:
- Coach profile photo and bio with real information
- Client transformation stories with verified results
- Media mentions with actual press features or partner logos
- All placeholder images with high-quality photography

## Getting Started

Simply open `index.html` in any modern web browser. No build process or dependencies required.

### Local Development

```bash
# Start a simple HTTP server
python3 -m http.server 8080

# Then visit http://localhost:8080 in your browser
```

## Customization

All styling is contained within the `<style>` tag in `index.html`. Key customization points:

- **Colors:** Modify CSS variables at the top of the style block
- **Typography:** Update Google Fonts imports and font-family declarations
- **Content:** Edit HTML text directly in the file
- **Images:** Replace placeholder divs with actual image tags
